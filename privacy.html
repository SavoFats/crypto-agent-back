Informativa sulla Privacy

Ultimo aggiornamento: 16 maggio 2026

1. Titolare del trattamento

Il titolare del trattamento dei dati personali è Zentra, raggiungibile all'indirizzo email saveriograssi1@gmail.com.

2. Dati raccolti

Raccogliamo le seguenti categorie di dati personali:

  • Dati di account: indirizzo email, nome utente, password (conservata in forma crittografata con bcrypt)
  • Credenziali API exchange: chiavi API Revolut X o Coinbase, conservate crittografate con AES-256 nei nostri server; non vengono mai trasmesse in chiaro
  • Dati di pagamento: gestiti interamente da Stripe. Non conserviamo numeri di carta o dati bancari. Riceviamo da Stripe un identificativo cliente (customer_id) e lo stato dell'abbonamento
  • Dati Telegram: chat ID del tuo account Telegram, se scegli di collegarlo per le notifiche
  • Dati di utilizzo: numero di sessioni di trading, configurazioni di sessione, log delle operazioni eseguite
  • Dati tecnici: indirizzo IP (usato per rate limiting e sicurezza), log di accesso

3. Finalità del trattamento

I tuoi dati vengono trattati esclusivamente per:

  • Erogare il servizio di trading automatizzato
  • Gestire il tuo account e l'autenticazione
  • Processare i pagamenti degli abbonamenti tramite Stripe
  • Inviare notifiche di trading via Telegram (solo se collegato)
  • Garantire la sicurezza della piattaforma e prevenire abusi
  • Adempiere a obblighi legali

4. Base giuridica

Il trattamento è basato su:

  • Esecuzione del contratto (art. 6.1.b GDPR): per erogare il servizio sottoscritto
  • Legittimo interesse (art. 6.1.f GDPR): per la sicurezza della piattaforma e la prevenzione delle frodi
  • Obbligo legale (art. 6.1.c GDPR): per adempiere a normative applicabili

5. Terze parti

I tuoi dati possono essere condivisi con:

  • Stripe, Inc. — processore di pagamento. Privacy policy Stripe
  • Railway Corp. — hosting del backend e del database. I server sono localizzati nell'UE o negli USA con garanzie adeguate
  • Telegram Messenger — per le notifiche, se collegato. Solo il chat ID viene trasmesso
  • Revolut / Coinbase — le richieste di trading vengono inviate alle API delle exchange tramite le tue credenziali

Non vendiamo né cediamo i tuoi dati a terzi per finalità di marketing.

6. Conservazione dei dati

I dati vengono conservati per il tempo necessario all'erogazione del servizio. Alla cancellazione dell'account, i dati vengono eliminati entro 30 giorni, ad eccezione dei dati contabili che devono essere conservati per obblighi fiscali (10 anni).

7. Sicurezza

Adottiamo misure tecniche e organizzative adeguate per proteggere i tuoi dati, tra cui:

  • Crittografia AES-256 per le chiavi API delle exchange
  • Hashing bcrypt delle password
  • Comunicazioni cifrate via HTTPS/TLS
  • Rate limiting e protezione contro attacchi brute force
  • Accesso ai dati limitato al personale autorizzato

8. I tuoi diritti (GDPR)

In qualità di interessato, hai diritto a:

  • Accesso: ottenere conferma del trattamento e copia dei tuoi dati
  • Rettifica: correggere dati inesatti o incompleti
  • Cancellazione: richiedere l'eliminazione dei tuoi dati ("diritto all'oblio")
  • Portabilità: ricevere i tuoi dati in formato strutturato e leggibile da macchina
  • Opposizione: opporti al trattamento basato sul legittimo interesse
  • Limitazione: richiedere la limitazione del trattamento in determinati casi

Per esercitare i tuoi diritti, scrivi a saveriograssi1@gmail.com. Risponderemo entro 30 giorni. Hai inoltre il diritto di proporre reclamo al Garante per la protezione dei dati personali (garanteprivacy.it).

9. Cookie

Zentra non utilizza cookie di profilazione o di tracciamento di terze parti. Vengono utilizzati esclusivamente token di autenticazione locali (localStorage) necessari al funzionamento del servizio.

10. Modifiche

Ci riserviamo di aggiornare questa informativa. Le modifiche sostanziali verranno notificate tramite l'app. L'uso continuato del servizio dopo la notifica costituisce accettazione delle modifiche.

Per qualsiasi domanda relativa al trattamento dei tuoi dati contatta: saveriograssi1@gmail.com
