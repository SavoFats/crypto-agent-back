Crea le API Key su Revolut X
Hai bisogno di due cose: un Key ID e un file private_key.pem. Si generano in pochi clic dal sito di Revolut X.
1
Apri Revolut X dal browser (non dall'app)
2
Vai in Account → API Keys → Create API Key
3
Dai un nome alla chiave (es. Zentra) e clicca Generate
4
Revolut ti mostra il Key ID e ti fa scaricare il file private_key.pem — tienili a portata di mano
💡 Il file .pem viene scaricato una volta sola al momento della creazione — non perderlo.

Il Key ID è una stringa corta che trovi sulla pagina delle API Key di Revolut X, accanto al nome che hai scelto.

Legenda segnali
I pallini colorati indicano le condizioni che l'agente verifica prima di aprire un trade.
1H
Trend 1h
EMA20 > EMA50 sull'oraria. Struttura di trend delle ultime 50 ore. È il filtro principale: niente long contro il trend strutturale.
15M
Trend 15m
EMA20 > EMA50 su 15 minuti. Conferma che il trend di medio termine è allineato con quello orario.
SLP
Slope EMA20
La EMA20 su 5m è in salita negli ultimi 15 minuti. Una EMA piatta o in discesa segnala trend in indebolimento — anche se è ancora sopra la EMA50, il momentum sta venendo meno. Solo slope positive vengono considerate.
PULL
Pullback
L'ultima candela chiusa è tornata vicino all'EMA20 su 5m (entro la soglia). Confermato su candela completamente chiusa — non sul prezzo live che può toccare EMA20 per un secondo durante un crollo.
RSI
RSI
RSI(14) in zona pullback (40–58). Troppo alto = coin già ipercomprata, non è un vero pullback. Troppo basso = crollo in corso, non un rimbalzo.
BODY
Candela bullish
L'ultima candela chiusa è verde e con corpo solido (almeno 30% del range totale). Esclude doji e candele rosse — segnali che il mercato non ha ancora deciso la direzione o sta scendendo.
VOL
Volume pullback
Il volume dell'ultima candela è sotto la media delle ultime 20 candele. Un pullback silenzioso (basso volume) indica che è una pausa normale del trend — non un dump con forti vendite. Volume alto su pullback = segnale di pericolo.
R
Rischio/Rendimento
Stop loss calcolato su 1.5× ATR e minimi recenti di candele chiuse. Se lo stop è troppo stretto o troppo largo rispetto ai limiti configurati, il trade viene scartato.
Verde = condizione soddisfatta  ·  Rosso = condizione non soddisfatta
L'agente entra solo quando tutti e 8 i segnali sono verdi contemporaneamente.
