Collega Revolut X
Revolut X collegato

Le chiavi API sono salvate e crittografate. L'agente utilizzerà Revolut X per eseguire gli ordini in modalità reale.

Passo 1 di 3
Genera le chiavi sul tuo computer
Revolut X usa crittografia Ed25519 — devi generare tu le chiavi localmente con due comandi nel terminale.
1
macOS — apri il Terminale (Cmd+Spazio → cerca «Terminale») ed esegui questi comandi uno alla volta, nell'ordine:
① Installa Homebrew
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
② Attiva Homebrew nella sessione corrente
eval "$(/opt/homebrew/bin/brew shellenv zsh)"
③ Installa OpenSSL
brew install openssl
④ Vai sul Desktop e genera le chiavi
cd ~/Desktop
$(brew --prefix openssl)/bin/openssl genpkey -algorithm ed25519 -out private.pem
$(brew --prefix openssl)/bin/openssl pkey -in private.pem -pubout -out public.pem
2
Trovi due file nuovi sul Desktop: private.pem e public.pem. Tienili a portata di mano.
🔒 private.pem è segreto. Non condividerlo mai. Zentra lo usa solo per firmare le richieste sul tuo dispositivo — non viene mai inviato ai nostri server in chiaro.
Registra la chiave su Revolut X
1
Vai su exchange.revolut.com → Profile → API Keys (da browser, non dall'app)
2
Apri public.pem con un editor di testo (tasto destro → Apri con → TextEdit su Mac, Blocco Note su Windows) e incolla tutto il contenuto incluse le righe BEGIN/END
3
Revolut genera il tuo API Key — una stringa di 64 caratteri. Copiala e incollala qui sotto:
Es: M1VKFtwB0M9C9QJO7goPlwrOytrJsSNE19txsmpsWIKz7xYu3f8aNucIyynAhYBy

Legenda segnali
I pallini colorati indicano le condizioni che l'agente verifica prima di aprire un trade.
1H
Trend 1h
EMA20 > EMA50 sull'oraria. Struttura di trend delle ultime 50 ore. È il filtro principale: niente long contro il trend strutturale.
15M
Trend 15m
EMA20 > EMA50 su 15 minuti. Conferma che il trend di medio termine è allineato con quello orario.
PULL
Pullback 15m
Il prezzo è vicino all'EMA20 su 15m entro la tolleranza configurata. Invece di inseguire un breakout già partito, il bot entra durante la correzione verso la media — prezzo migliore, stop più stretto, più upside davanti.
RSI
RSI
RSI(14) in zona neutrale (40–65). Troppo alto = coin già ipercomprata, poco spazio per salire. Troppo basso = possibile falso segnale in downtrend.
BODY
Candela bullish
L'ultima candela chiusa è verde e con corpo solido (almeno 30% del range totale). Esclude doji e candele rosse — segnali che il mercato non ha ancora deciso la direzione o sta scendendo.
VOL
Volume rimbalzo
Il volume dell'ultima candela è sopra la media delle ultime 20 candele. Un rimbalzo con volume alto conferma che c'è partecipazione reale — bassa partecipazione sul pullback è spesso un falso segnale.
R
Rischio/Rendimento
Stop loss calcolato su 1.5× ATR e minimi recenti di candele chiuse. Se lo stop è troppo stretto o troppo largo rispetto ai limiti configurati, il trade viene scartato.
Verde = condizione soddisfatta  ·  Rosso = condizione non soddisfatta
L'agente entra solo quando tutti e 8 i segnali sono verdi contemporaneamente.
